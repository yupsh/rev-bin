# yup-rev

```
NAME:
   rev - reverse lines characterwise

USAGE:
   rev [OPTIONS] [FILE...]

      The rev utility copies the specified files to standard output, reversing
      the order of characters in every line. If no files are specified, standard
      input is read.

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --separate, -s  reverse characters separately for each line (default: false)
   --help, -h      show help
```
